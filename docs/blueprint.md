# **App Name**: Pawsh Cuts

## Core Features:

- Service Listing: Display a list of grooming services offered for cats and dogs.
- Stylish Pricing: Present prices in interactive cards, enhancing the pricing user experience.
- Appointment Scheduling: Allow users to book grooming appointments through a user-friendly form.
- Before & After Gallery: Showcase grooming results through a before-and-after photo gallery with interactive transitions.
- Image Enhancement: Generative AI tool: Enhance pet photos before displaying them in the gallery to maintain a high visual standard.
- Interactive Paw Animations: Display small paw animations when the user hovers the elements

## Style Guidelines:

- Primary color: Warm Teal (#5FB3B3) for a calming and welcoming feel.
- Secondary color: Coral Pink (#F28BA8) for soft highlights and accents.
- Accent color: Lavender Purple (#9B8FE0) for CTAs and interactive elements.
- Background: Warm Peach Cream (#FFF1E6) to evoke a cozy and premium ambiance.
- Headings font: 'Poppins', a geometric sans-serif font, is recommended for the headings due to its clean and modern look, appropriate for headlines and short amounts of body text; if longer text is anticipated, use this for headlines and 'Inter' for body. Note: currently only Google Fonts are supported.
- Body text font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look, suitable for body text. Note: currently only Google Fonts are supported.
- Use cute and simple line icons for services, navigation, and interactive elements, focusing on paw prints and grooming tools.
- Maintain a mobile-first, responsive layout with soft edges and rounded corners to enhance the friendly, approachable feel.
- Employ subtle paw icon patterns as background textures on specific pages for a playful touch. Paw color: slightly darker peach #F7D8C8.
- Implement refined micro-interactions such as hover effects, tap feedback, and smooth transitions to enhance user engagement.
